# KHKopernicusUtilities
A set of tools to streamline KSP Planetmodding.

## Currently available tools:

- Convert Declination and Right Ascension to Inclination and Longitude of Ascending Node (for placement of real star systems)
- Convert Latitude and Longitude to Cartesian (for PQS City and KSC placement)
- invWaveLength calculator (for stock atmosphereFromGround)
- More coming soon, hopefully
